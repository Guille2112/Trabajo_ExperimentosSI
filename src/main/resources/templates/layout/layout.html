C.A.I.

C.A.I. Es una aplicación web diseñada para ayudar al registro de insumos para tiendas, con el fín de tener un inventario apropiadamente organizado y una adecuada lista de quienes interactuan con estos insumos.

Funciones

  • Registro de empleados
  • Asignación de Lista de Compras
  • Registro de Proveedores
  • Registro de Recursos
  • Reportes de Incidencias
